- 투명한 방법으로 정보를 동기화
- 디스크립터를 사용함으로서 재사용성 또한 높아질 수 있음
- dry 원칙 고려 !

### brog 패턴
- 너 정말 싱글톤 사용하고 싶어!? 그럼 brog 패턴
- 여러개의 인스턴스는 하나의 상태를 공유한다
```python3
class Borg:
    _shared_state = {}

    def __init__(self):
        self.__dict__ = self._shared_state

a = Borg()
b = Borg()
print (a is b) #False
a.site = 'jaeyoung'
print(b.site) # jaeyoung
```
